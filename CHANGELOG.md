# Changelog

All notable changes to DataBees will be documented in this file.

## [0.1.0] - 2024-07-07

### Added
- Initial release
- Support for MySQL, PostgreSQL, MongoDB, SQLite, MSSQL
- Connection management with unlimited connections
- Query execution with results display
- Export to CSV, JSON, XLSX
- Basic encryption for credentials
- VSCode integration
- Stripe payment integration
- 30-day free trial

### Coming Soon
- Parquet export support
- Git integration
- Saved queries
- Query history
- Team collaboration
- Advanced filtering and search
