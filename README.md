# DataBees - Premium VSCode Database Client

![DataBees](assets/databees-banner.png)

🐝 **DataBees** is a powerful, modern database client extension for VSCode that lets you manage unlimited database connections with enterprise-grade security.

## ✨ Features

- 🔗 **Unlimited Connections** - Connect to multiple databases simultaneously
- 📑 **Unlimited Tabs** - Work with multiple queries at once
- 💾 **Multi-Format Export** - CSV, XLSX, JSON, Parquet, and more
- ☁️ **Cloud Database Support** - AWS RDS, Google Cloud SQL, Azure Database, Heroku
- 🔐 **Enterprise Security** - AES-256 encryption for stored credentials
- 🚀 **Lightning Fast** - Optimized for performance
- 📊 **Rich Query Results** - Advanced filtering and visualization
- 🎯 **30-Day Free Trial** - $1/month after trial

## 🗄️ Supported Databases

- MySQL / MariaDB

## 🚀 Quick Start

### Installation

1. Install the extension from VSCode Marketplace
2. Open the Command Palette (`Ctrl+Shift+P`)
3. Type "DataBees: Open Database Client"
4. Start creating connections!

### Create Your First Connection

1. Click the DataBees icon in the Activity Bar
2. Click "New Connection"
3. Select your database type
4. Fill in connection details
5. Test connection
6. Save!

## 📖 Usage

### Execute Queries

```sql
SELECT * FROM users WHERE status = 'active';
```

### Export Results

Right-click on results table and select:
- **Export as CSV**
- **Export as XLSX**
- **Export as JSON**
- **Export as Parquet**

### Manage Connections

- View all connections in the sidebar
- Edit connection details
- Duplicate connections
- Test connections
- Delete connections

## 🔒 Security

DataBees uses industry-standard security measures:

- 🔑 **AES-256 Encryption** - All credentials encrypted at rest
- 🗝️ **VSCode Secret Storage** - Credentials stored in system keychain
- 🛡️ **SSL/TLS Support** - Secure connections to cloud databases
- ✅ **No Remote Telemetry** - Your data stays local
